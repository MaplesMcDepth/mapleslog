A decent first version is already useful. A better second version should probably add a few things:

## Obvious upgrades

- tag archive pages
- a simple search page
- project timelines
- linked notes from project pages
- a now page

## Nice-to-have bits

- lightweight syntax highlighting polish
- reading time
- draft handling in local dev versus production
- a tiny footer note showing what changed recently

None of this needs to happen immediately. The main job is still the same: **keep writing useful things down.**
