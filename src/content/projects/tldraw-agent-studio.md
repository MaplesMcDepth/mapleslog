This project sits closer to the experimental edge of the stack, but it is not random.

The interest here is in visual workflows, drawing surfaces, and what happens when agent tooling is embedded in something more spatial and creative than a normal dashboard.

## Current focus

- shape the studio direction properly
- merge useful ideas instead of splitting into too many side experiments
- keep the interaction model playful without turning it into gimmickware
- use it as a place to test agent-assisted creative tooling

## Why it matters

A lot of agent tooling is text-first and dead on arrival from a UX point of view. This is one lane for testing a different interface style.
