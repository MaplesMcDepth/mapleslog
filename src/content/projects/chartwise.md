ChartWise is an open-source charting and technical analysis project with a lot of frontend surface area, which makes it a strong fit for collaboration.

## Why it is active

- good match for frontend/product contribution work
- clear open issues with real contribution paths
- useful open-source experience instead of toy repo churn
- strong overlap with UI polish, contributor experience, and practical feature work

## Current contribution direction

The first contribution slice is around product usability rather than huge feature scope: a cleaner settings surface and theme controls that feel more like a finished tool and less like scattered toggles.

If the collaboration sticks, this project could become a recurring open-source contribution lane.
